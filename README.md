# RoamStay

RoamStay is a full-stack web application that allows users to search for places to stay, view property details, and book accommodations, while hosts can list and manage their own properties. The project focuses on building a real-world booking platform with complete frontend, backend, and database integration.

This project was developed as an academic full-stack web development project to gain hands-on experience with frontend, backend, and database integration.

---

## ✨ Features

### User Features
- Browse and search available properties
- View detailed property information
- Add review and reating to available property
- Book accommodations

### Host Features
- Create new property listings
- Edit existing listings
- Delete listings

---

## 🛠 Tech Stack

- **Frontend:** HTML, CSS, JavaScript, EJS Templates
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose)  
- **Version Control:** Git & GitHub

## 🚀 Getting Started

### Prerequisites
- Node.js
- MongoDB

### Installation

```bash
git clone https://github.com/ShreyaVispute021/RoamStay.git
cd RoamStay
npm install
```

## Run Application
```bash
nodemon app.js
```
The application will be available at:
👉 http://localhost:8080
